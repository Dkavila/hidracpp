O QUE SÃO MNEMÔNICOS:


As instruções na programção de baixo nivel são representadas por números, os OP CODES. Para o programador os OP CODES não são intuitivos, por isso quando estamos escrevendo o código assembly existe o recurso de usar mnemônicos, abreviações de palavras que remetem ao sentido da expressão, que serão transformadas em seus respectivos OP CODES pelo montador(assembler) na hora de gerar o código em linguagem de máquina.

0 - NOP(No Operator)


A instrução 0-NO OPERATOR não realiza nenhuma operação apenas indica a falta de dados ou instruções em um endereço. Por default todos os endereços no arquivo em branco valem NOP.

16 - STA(Store Adress)


A instrução 16 - STORE ADRESS e deve ter o formato:
STA endereço
A instrução tem como função armazenar o valor do acumulador no endereço de memoria fornecido sobrescrevendo qualquer valor antes contido no endereço. A instrução não altera o valor do acumulador.

32 - LDA(Load Adress)


A instrução 32 - LOAD ADRESS e deve ter o formato:
LDA endereço
A instrução tem como função buscar o valor armazenado no endereço e carrega-la no acumulador, sobrescrevendo qualquer valor anterior ali contido.

48 - ADD(Adição)


A instrução 48 - ADIÇÃO deve ter o formato:
ADD endereço
A instrução tem como função fazer a soma bit a bit do número contido no endereço de memória com o acumulador e salvar o resultado no acumulador.


CUIDADO!NA MÁQUINA NEANDER A SOMA BIT A BIT NÃO CUIDA SE O RESULTADO ULTRAPASSA A FAIXA DE REPRESENTAÇÃO OU SE HÁ ESTOURO E NA MÁQUINA AHMES AS FLAGS B(BORROW) V(OVERFLOW) C(CARRY) AVISARÃO CASO OCORRA, MAS FARÃO A SOMA BIT A BIT IDEPENDENTE DISSO.

OR endereço
A instrução tem como função fazer o "ou lógico" bit a bit do número contido no endereço de memória com o acumulador e salvar o resultado no acumulador.

AND endereço

NOT

128 - JMP(Jump)

.
A instrução 128 - jump deve ter o formato:
JMP endereço
A instrução tem como função desviar incondicionalmente a execução para uma certa área de memória mudando o valor do Program Counter para o endereço fornecido.
