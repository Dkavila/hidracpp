REGISTARDORES

O QUE SÃO REGISTARDORES:


Registradores são unidades de memória capazes de armazenar n bits, e conseguem deslocar e armazenar informação binária. São o meio mais rapido e mais carode se armazenar informação.

Lista de Registradores:

  • PC(Program Counter)
  • AC(Acumulator)
  • Registradores de Estado
    • N
    • Z
    • V
    • C
    • B
